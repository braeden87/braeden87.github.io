Random Quote Generator Project

Languages: JavaScript, PHP, HTML, CSS
Technologies / Frameworks: API's, Database

In this project I utilized the forismatic API to generate random quotes. Users have the option to cycle through a list of random quotes and even post them on Twitter. Users also have the option to create an account on this site and then save their favorite quotes so that they can view them later.

Calculator Project

Languages: JavaScript, HTML, CSS

This project is my take on a simple calculator. You have the option to work in either dark or light mode, and all of your calculations will be displayed in the past calculations column. If you leave the page and then come back to it, your preference for dark or light mode will automatically be initialized.
